\subsection{Information Theory}

Recall that in Shannon Information theory (see Ch. \ref{ch-shannon-info}), we define
for random variables $\rvx, \rvy$
\begin{itemize}

\item the {\bf entropy}
\beq
H(\rvx)=-\sum_{x\in val(\rvx)} P(x)\ln P(x)
\eeq

\item the {\bf joint entropy}
\beq
H(\rvx, \rvy) =-
\sum_{x\in val(\rvx)}
\sum_{y\in val(\rvy)}
P(x,y)\ln P(x, y)
\eeq

\item the {\bf conditional information}
\beqa
H(\rvy\mid \rvx) &=&-
\sum_{x\in val(\rvx)}
\sum_{y\in val(\rvy)}
P(x,y)\ln P(y\mid x)
\\
&=&
H(\rvx, \rvy)-H(\rvx)
\eeqa

\item the {\bf mutual information (MI)}
\beqa
H(\rvx:\rvy)&=&\sum_{x\in val(\rvx)}
\sum_{y\in val(\rvy)}
P(x,y)\ln \frac{P(x,y)}{P(x)P(y)}
\\
&=& H(\rvy)-H(\rvy\mid\rvx)=H(\rvx)-H(\rvx\mid\rvy)
\eeqa
\end{itemize}

Here are some standard inequalities
that we will find useful in the rest of this chapter.
\begin{claim}
\beq
0\leq H(\rvx)\leq \ln|val(\rvx)|
\eeq

\beq
0\leq H(\rvy\mid \rvx)\leq H(\rvx, \rvy)
\eeq

\beq
0\leq H(\rvx: \rvy)\leq \min(H(\rvx), H(\rvy))
\label{eq-MI-bounds-rg}
\eeq
\end{claim}
\proof
Left to the reader.
\qed

We will use MI to
measure the correlation
between random variables $\rvx$ and $\rvy$.

Note that MI achieves its zero
lower bound
when $P(x,y)=P(x)P(y)$
(i.e., $\rvx$ and $\rvy$ are uncorrelated).

Furthermore, MI achieves its 
upper bound given by Eq.(\ref{eq-MI-bounds-rg})
if either 

\begin{itemize}
\item
$\rvy$
is a deterministic
function of $\rvx$; i.e., 

\beq P(y\mid x)=\indi(\rvy=f(\rvx))\eeq
In that case,
$\ln P(y\mid x)$ is either $0\ln(0)$
or $\ln(1)$ so 

\beq
H(\rvy\mid\rvx)=0
\eeq
Furthermore,
\beq
H(\rvy\mid\rvx)=0\implies 
 H(\rvx: \rvy)= H(\rvy)
\eeq

\item $\rvx$ 
is a deterministic function of $\rvy$; i.e.,  
 
\beq
P(x\mid y)=\indi(\rvx = g(\rvy))
\eeq
In that case,
$\ln P(x\mid y)$ is either $0\ln(0)$
or $\ln(1)$ so

\beq
H(\rvx\mid\rvy)=0
\eeq
Furthermore

\beq
H(\rvx\mid\rvy)=0\implies 
 H(\rvx: \rvy)= H(\rvx)
\eeq
\end{itemize}




Define the {\bf MI Efficiency} $\eps(\rvy\mid\rvx)$ by
\beq
\eps(\rvy\mid\rvx)=
\frac{H(\rvy: \rvx)}{H(\rvy)}
= 1 - \frac{H(\rvy\mid \rvx)}{H(\rvy)}
\eeq
From $0\leq H(\rvy:\rvx)\leq H(\rvy)$ we get

\beq
0\leq \eps(\rvy\mid\rvx)\leq  1
\eeq

From its definition, we see that the MI efficiency has a 
singularity of the type 0/0
when $H(\rvy\mid \rvx)= H(\rvy)=0$.

We will henceforth
refer to the
$(H(\rvy), H(\rvy\mid\rvx))$ plane
as the {\bf entropy versus conditional info (E-CI) plane}


It is instructive (see Fig.\ref{fig-eps-sing})
to plot in the E-CI plane, the loci
on which the efficiency
equals certain values.

that I find instructive

The standard Ising model
is defined on a cubic lattice,
as shown in Fig.\ref{fig-ising-1-2}.

Let


$Bool=\bool$

 
$S_i\in \{-1, +1\} =2 Bool-1=\ZZ_\pm$

$T=$ temperature

$
\beta = 1/(k_B T)
$. We will set Boltzmann constant $k_B$  to 1.

$D=$ dimension of lattice. We refer to $D=1$ as 1-dim, and to $D=2$ as 2-dim.

$NN(i)=$ set of indices of nearest neighbors 
to $S_i$.

$z=2D$= {\bf coordination number},
number of nearest neighbors
internally in the lattice, away from the boundary. 
$z=2$ in $D=1$ and $z=4$ in $D=2$.
For spins on the boundary, the number of NN
is less than $z$. For example, as you 
can see in Fig.\ref{fig-ising-1-2}, for a rectangular lattice 
in 2-dim, the corner sites have 2 NN, and the non-corner
boundary sites have 3 NN. Only the internal sites have 4 NN.

Assume coupling constants $J, h\geq 0$.

The {\bf D-dim Ising Model} is defined as follows.

For $i=1, 2,3, \ldots, n$, 
define {\bf energies} $\cale_i$ and $\cale$ by
\beq
\cale_i = \left
[-J S_{i}\sum_{j\in NN(i)}S_j \right]- h S_{i},
\quad
\cale=\sum_{i=1}^n\cale_i
\eeq

and a 
Partition Function $Z$ by
\beq
Z=\left[\prod_{i=1}^n\sum_{S_i\in\ZZ_\pm}
\right]
e^{-\beta \cale}
\eeq
Let

$\xi=$ {\bf correlation length}. $\av{\rvS_i, \rvS_{i+r}}\sim e^{-r/\xi}$. $\xi\rarrow \infty$ at critical points

$\av{m}=\frac{1}{n}\sum_{i=1}^n\rvS_i=$ {\bf magnetization}

$T_c= 1/\beta_c=$ {\bf Curie Temperature} (a.k.a. critical temperature). Magnetization is non zero for $T< T_c$ (spontaneous magnetization) and zero for $T>T_c$ (disordered phase).

The 2-dim Ising model with $h=0$ has a critical
point at $T=T_c$.
If either $D=1$ or $h\neq 0$,
 there are no critical 
points at any finite temperature, just phase transitions.



Ising Model can only be solved
exactly for $D=1, 2$. 
For other $D$, must solve numerically.

For the 2-dim Ising model with $h=0$:

\begin{itemize}
\item Curie Temperature

  \beq
  T_c = \frac{2J}{\ln(1+\sqrt{2})}
  \eeq

\beq
\beta_c J = \frac{1}{2}\ln(1 +\sqrt{2})
\approx 0.4407
\eeq

\item correlation length

\beq
\xi \sim |T-T_c|^{-\nu},\quad \nu=1
\eeq


\end{itemize}












\section{Scaling a General Bnet}

Consider a general
bnet with nodes $\rvx_i$
for $i=1,2, \ldots, n$. Let

$\calr=$ set of root nodes. 

$\calr^c=$ set of non-root nodes.

$x^{:n}= (x_1, x_2, \ldots, x^n)$

Then

\beq
P(x^{:n}) = \left[ \prod_{\rvx_i\in \calr^c}P(x_i\mid pa(\rvx_i))\right]\prod_{\rvx_i\in\calr}P(x_i)
\label{eq-rg-full-bnet-distr}
\eeq

For $1<b<\infty$, let
\beq
P_b(x_i\mid pa(\rvx_i))=
\frac{P(x_i\mid pa(\rvx_i))^b}
{Z_i(b)}=\pi_i(b)
\eeq
where

\beq
Z_i(b)=\sum_{x_i\in val(x_i)}
P(x_i\mid pa(\rvx_i))^b
\eeq

\begin{mdframed}[hidealllines=true,backgroundcolor=blue!10]
{\bf  As $b\rarrow \infty$,}

 $(\pi_1, \pi_2, \ldots, \pi_n)$ tends to a one hot vector; i.e., all its components tend to zero except for one component, the one that is smallest at $b=1$, which tends to 1:

\beq
\pi_i(b)\rarrow \delta(i, i_0),\quad \text{ where } i_0=\argmin_i \pi_i(b=1)
\eeq
Therefore, $H(\rvx_i\mid pa(\rvx_i))$
(for $P_b$)
tends to a sum of terms of the form $0\ln(0)$
or $\ln(1)$. Thus

\beq
H(\rvx_i\mid pa(\rvx_i))\rarrow 0
\eeq
Thus

\beq
H(\rvx_i:pa(\rvx_i))\rarrow
H(\rvx_i)
\eeq

If $H(\rvx_i)$ remains finite as $b\rarrow \infty$, then
$\eps(\rvx_i \mid pa(\rvx_i))\rarrow 1$,
but if $H(\rvx_i)$ tends to zero, then 
$\eps(\rvx_i \mid pa(\rvx_i))$ tends
to a 0/0 singularity at $H(\rvy\mid \rvx)
= H(\rvy)=0$
\end{mdframed}

\begin{claim}\label{cl-mutual-info-bd}

\beq
\sum_{i=1}^n H(\rvx_i)-H(\rvx^{:n})=
\sum_{\rvx_i \in \calr^c} H(\rvx_i : pa(\rvx_i))
\eeq
Note that both sides equal zero
when all the nodes are uncorrelated.
\end{claim}
\proof

\beq
H(\rvx_i : pa(\rvx_i))=
\sum_{x_i}\sum_{pa(\rvx_i)}P(x_i, pa(\rvx_i))
\ln \frac{P(x_i\mid pa(\rvx_i))}{P(x_i)}
\eeq

\beq
H(\rvx_i)=-\sum_{x_i}P(x_i)\ln P(x_i)
\eeq

\beq
H(\rvx^{:n})=-\sum_{x^{:n}}P(x^{:n})\ln P(x^{:n})
\eeq

From Eq.(\ref{eq-rg-full-bnet-distr}), we get
\beq
\frac{P(x^{:n})}{\prod_{i=1}^n
P(x_i)}=\prod_{\rvx_i \in \calr^c}  \frac{P(x_i\mid pa(\rvx_i))}{P(x_i)}
\eeq
Hence

\beq
\sum_{x^{:n}}P(x^{:n})\ln \frac{P(x^{:n})}{\prod_{i=1}^n P(x_i)}=
\sum_{\rvx_i \in \calr^c}H(\rvx_i : pa(\rvx_i))
\eeq

\qed

This last claim says that the \qt{bound entropy} (i.e., the
entropy between when $n$ random variables 
are independent and when they are correlated) equals the
sum of the mutual informations for 
the non-root nodes.

\begin{claim}
\beq
\max_i H(\rvx: \rva_i) \leq H(\rvx:\rva_1, \rva_2, \ldots, \rva_m)
\leq \sum_{i=1}^m H(\rvx: \rva_i)
\label{eq-mi-right-bd}
\eeq

\end{claim}
\proof

We will prove this for $m=3$ and leave 
the general case to the reader.

Averaging the logarithm of the following chain rule for conditional
probabilities

\beq
\frac{P(x, a_1, a_2, a_3)}{P(x)P(a_1, a_2, a_3)}=
\left[\frac{P(x, a_1\mid a_2, a_3)}{P(x\mid a_2, a_3)P(a_1\mid a_2, a_3)}\right]
\left[\frac{P(x, a_2\mid a_3)}{P(x\mid a_3)P(a_2\mid a_3)}\right]
\frac{P(x, a_3)}{P(x)P(a_3)}
\eeq
we get

\beq
H(\rvx: \rva_1, \rva_2, \rva_3)=
H(\rvx: \rva_1 \mid\rva_2, \rva_3)
+ H(\rvx:\rva_2\mid\rva_3)
+ H(\rvx:\rva_3)
\label{eq-chain-rule-x-a123}
\eeq
Both sides of the inequality  Eq.(\ref{eq-mi-right-bd})
follow from Eq.(\ref{eq-chain-rule-x-a123}). Indeed,
\beq
H(\rvx: \rva_1, \rva_2, \rva_3)=
\underbrace{
\underbrace{H(\rvx: \rva_1 \mid\rva_2, \rva_3)}_{\leq H(\rvx: \rva_1)}
+ \underbrace{H(\rvx:\rva_2\mid\rva_3)}_
{\leq H(\rvx:\rva_2)}
+ H(\rvx:\rva_3)}_{\geq H(\rvx:\rva_3)}
\eeq
\qed


Dividing Eq.(\ref{eq-mi-right-bd}) by $H(\rvx)$ 
and identifying the $\rva_i$ nodes with the parents $pa(\rvx)$
of $\rvx$ , we get
\beq
\max_{\rva\in pa(\rvx)}\eps(\rvx| \rva) \leq \eps(\rvx\mid pa(\rvx))
\leq \sum_{\rva\in pa(\rvx)} \eps(\rvx\mid \rva)
\eeq

\subsection{Converting 1-time bnet to a dynamical one}

The 1-time Ising bnet presented so far does not
equilibrate. To build a bnet that does
equilibrate, we can make many copies 
(\qt{times-slices}) of the above 1-time bnet
and connect them to make a dynamic bnet.

Add a functional dependance $(t)$ to all the random variables $\rvD_i$ in the 1-time bnet presented above. Do the same with $(t+1)$. Now we have two subsequent time slices.

To connect them, set $S_i^X(t+1)$ equal to $S_i^Y(t)$ for all $i$. In other words,
we need to add the following TPM

\beq\color{blue}
P(S_i^X(t+1)\mid S_i^Y(t))=\quad
\indi(S_i^X(t+1)=S_i^Y(t))
\eeq
for all $i$.

Conclusions

1. Our 2-dim Ising-dbnet model exhibits a discontinuity in the magnetization at the same Curie temperature as the standard vanilla 2-dim Ising model.

2. For $p0\neq .5$, the parametric curves flow smoothly as $\beta\rightarrow \infty$, towards $(H(y), H(y\mid x))=(0,0)$, where the efficiency is discontinuous.

4. For $p0=.5$, the parametric curves flow smoothly as $\beta\rightarrow \infty$, vertically down.
