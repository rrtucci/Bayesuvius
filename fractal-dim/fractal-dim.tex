\chapter{Fractal Dimension}
\label{ch-fractal-dim}

This chapter is based on discussions with an AI and on
Ref. \cite{wiki-fractal}.

The {\bf fractal dimension} of an object
can be understood as 
a metric for a type of tree (we will call it the object's {\bf fractal tree}), 
and trees can be understood as a type of bnet. 

Let

$N=$  number of branches
that each tree branch splits into, number of pieces that each piece breaks into.

$b=\frac{\ell_i}{\ell_f}=$ length $\ell_i$ (not area or volume) of initial piece
divided by length $\ell_f$ of final piece

$D=$ {\bf fractal dimension}, $D>0$ can be less than or greater than 1.

Define
\beq
b^D = \frac{\ell_i^D}{\ell_f^D} = N
\eeq
Hence, 
\beq
D = \frac{\ln N}{\ln b}
\eeq

A {\bf self-similar object} is
an object which looks the same regardless of the magnification you observe it at. In other words, it's an object 
for which the above formula for fractal
dimension
gives the same answer regardless of the magnification you measure $\ell$
initially at. For a self similar object, every node of the fractal tree (except the root node) is identical.


Examples:
\begin{itemize}
\item line segment (see Fig.\ref{fig-line-segment})

\beq
N=2, \quad\ell_i/\ell_f=2\implies\quad
D=\frac{\ln 2}{\ln 2}=1
\eeq



\item square (see Fig.\ref{fig-square})

\beq
N=4,\quad \ell_i/\ell_f=2\implies\quad
D=\frac{\ln 4}{\ln 2}=2
\eeq


\item Cantor set (see Fig.\ref{fig-cantor-set})

\beq
N=2,\quad \ell_i/\ell_f=3\implies\quad
D=\frac{\ln 2}{\ln 3}\approx 0.63
\eeq

\end{itemize}
