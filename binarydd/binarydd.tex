\subsection{Simplification of Binary Tree}


3 Rules

\begin{enumerate}
\item Merge equivalent leaves (EL)
\item Merge isomorphic nodes (IN)
\item Eliminate parallel 0/1 arrows (PA) by merging source
and target nodes of arrows.
\end{enumerate}
